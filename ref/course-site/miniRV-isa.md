# miniRV 指令集

miniRV是RV32I的子集，兼容RISC-V。包含32个通用寄存器和37条指令（R型10条、I型15条、S型3条、B型6条、U型2条、J型1条）。

## 通用寄存器

| 寄存器 | 助记符 | 释义 |
|--------|--------|------|
| x0 | zero | 常数0 (只读) |
| x1 | ra | 返回地址 |
| x2 | sp | 堆栈指针 |
| x3 | gp | 全局指针 |
| x4 | tp | 线程指针 |
| x5~x7 | t0~t2 | 临时变量 |
| x8 | s0/fp | 堆栈帧指针 |
| x9 | s1 | Saved Register |
| x10~x11 | a0~a1 | 参数/返回值 |
| x12~x17 | a2~a7 | 参数 |
| x18~x27 | s2~s11 | Saved Register |
| x28~x31 | t3~t6 | 临时变量 |

## 指令格式 (与RISC-V相同)

- R型: funct7[31:25] + rs2[24:20] + rs1[19:15] + funct3[14:12] + rd[11:7] + opcode[6:0]
- I型: imm[31:20] + rs1[19:15] + funct3[14:12] + rd[11:7] + opcode[6:0]
- S型: imm[31:25] + rs2[24:20] + rs1[19:15] + funct3[14:12] + imm[11:7] + opcode[6:0]
- B型: imm[31] + imm[30:25] + rs2[24:20] + rs1[19:15] + funct3[14:12] + imm[11:8,4:1] + opcode[6:0]
- U型: imm[31:12] + rd[11:7] + opcode[6:0]
- J型: imm[31,19:12,20,30:21] + rd[11:7] + opcode[6:0]
