# 数据通路设计

设计数据通路的基本思想：根据每条指令的功能描述和指令格式，推演出指令执行过程所需的功能部件及其输入输出信号之间的连接关系（指令级数据通路）。

## 方法：表格驱动设计

通过数据通路表来描述数据通路。对每条指令分析：
1. 需要哪些功能部件
2. 部件之间的连接关系
3. 各部件需要执行的具体功能 → 控制信号取值

对所有指令分析完成后，新增一行"综合"，整合所有部件的输入信号。

### 多输入源处理
- 添加多路选择器（MUX）解决多输入源问题
- 立即数的多格式问题：拓展SEXT输入位宽，将指令码中所有可能的立即数位输入到SEXT

最后将数据通路表图形化，画出CPU数据通路图。
